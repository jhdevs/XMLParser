Необходимо написать приложение для парсинга и отображения XML файла. XML файл заведомо большой (можно взять, например, этот: http://monolith.izrukvruki.ru/img/files/external/irr_ads.xml.gz - осторожно, весит около 30Мб).

Пользовательский интерфейс реализовать на SWING (достаточно будет кнопки "загрузить" и таблицы, где одна строка соответствует одному тегу offer, а столбцы - подтегам (url, date, mark и т.п.)). Файл должен парситься в realtime, т.е. по мере парсинга файла в таблице должны появляться новые строчки. На время парсинга файла должна появляться кнопка "Отмена".
Для парсинга XML использовать стандартные средства Java.
